Data Science Projects

Choosing between Toronto and Montreal: A Clustering Approach

This project is a data analysis tool for people looking to settle in one of the two largest cities in Canada. Under this project, you can find an easy-to-read report divided into two sections: (I) a descriptive statistics analysis to put both cities in the context of the other major Canadian cities, and (II) a machine learning analysis to find how similar or different are the neighborhoods of the two cities.

Web-Scraping and Data Visualization of my Fantasy-Soccer Mini-League

This project is a web scraping tool for my Fantasy Soccer Mini-league, which is a game I have been playing with very close longtime friends for more than ten years. This year I decided to level up my analysis of the game and create this interesting web-scraping project.

If you are not familiar with fantasy sports, they are about selecting players in the game's platform. Your team performs based on the real-life performance of the chosen players. Sure, you can choose players and get lucky; nevertheless, data analysis plays a large role in your performance.

For the past years, I haven't dedicated much time to the game. However, this is a great opportunity for a web-scraping project because data are all over the place. Scraping the data presented several challenges that this project tackles.

Choosing between Toronto and Montreal: A Clustering Approach

Description of the Project:

This project is a data analysis tool for people looking to settle in one of the two largest cities in Canada. Under this project, you can find an easy-to-read report divided into two sections: (I) a descriptive statistics analysis to put both cities in the context of the other major Canadian cities, and (II) a machine learning analysis to find how similar or different are the neighborhoods of the two cities.

In addition, you can find two GitHub repositories for each of the sections of the report. Please feel free to emulate my data collection, analysis, and visualization. Suggestions and improvement recommendations are always welcome.

The machine learning algorithm used to compare neighborhoods of the two cities is a K-means clustering technique. Overall, the purpose of this project is to guide Canadian immigrants (or people moving from one city to the other) and define if the type of neighborhood of each city is a factor to consider when deciding between living either in Toronto or Montreal.
